Installation
------------
- Copy the Monte dir (which should contain the .dll and Default settings) 
- Paste into the Assets folder of your Unity project

Using Monte
-----------
- Read the Monte Wiki for full details (https://github.com/cring002/Monte/wiki/)
- Create your own AIState
- Train a model using this AIState
- Use one of the agents provided and give it an AIState and the Model
- The AI will provide a 'next' state.
- Use this as your next AI state.
